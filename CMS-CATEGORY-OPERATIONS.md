# CMS 카테고리 운영 기준

이 문서는 피아노 사이트 CMS의 공개 블로그 분류를 고정한다. 카테고리 슬러그와 기존 URL은 임의로 추가·변경하지 않는다. 코드의 최종 기준은 `app/src/lib/content.ts`의 `CATEGORY_SEO`다.

## 키워드 역할 분담

- 메인 키워드 `피아노 레슨`은 홈(`/`)이 담당한다.
- 전환형 확장 키워드는 성인·어린이·방문·입시·비용 등 고정 서비스 랜딩이 담당한다.
- CMS 글은 정보 키워드와 대상·지역 롱테일만 발행한다.
- 각 글의 첫 번째 태그는 대표 목표 키워드이며 다른 공개 URL과 중복하지 않는다.

## 발행 규칙

1. 카테고리를 먼저 고른다. 관리자 화면은 해당 카테고리에 허용된 검색 의도와 토픽 클러스터만 보여 준다.
2. 메인·전환형 확장·탐색·정책 역할은 블로그에서 발행하지 않는다. 해당 키워드는 고정 랜딩에서 운영한다.
3. 카테고리의 대표 키워드를 개별 글의 대표 키워드로 재사용하지 않는다. 글은 더 구체적인 질문형·대상형·지역형 롱테일을 맡는다.
4. 지역 카테고리는 실제 지역명이 검색 의도의 중심인 글만 사용한다. 지역 제한이 없는 온라인 레슨 비교는 레슨 선택·비용으로 분류한다.
5. 카테고리나 글 슬러그를 바꾸면 기존 공개 주소를 `post_redirects`에 보존한다.

## 기존 글 정리

`온라인 피아노 레슨` 글은 지역 검색 글이 아니므로 `/blog/local/online-piano-lesson`에서 `/blog/lesson-guide/online-piano-lesson`로 이동한다. 기존 주소는 새 주소로 리디렉션되어 기존 검색 신호와 외부 링크를 보존한다.
